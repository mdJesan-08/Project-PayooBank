# 💳 PayooBank - JavaScript DOM Manipulation Project

A simple **digital banking interface** built to practice and strengthen **JavaScript DOM manipulation** concepts.  
This project simulates basic banking operations such as login, adding money, and cash out.

The main goal of this project was to **get hands-on experience with JavaScript and DOM manipulation** while building a small interactive web application.
